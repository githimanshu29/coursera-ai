## What is Learnova?

Learnova is a full-stack AI-powered learning platform that generates complete courses on any topic. Users can create a course layout, generate content all at once, or build chapter-by-chapter with RAG context. The platform includes curated video learning, quizzes, progress tracking, and a public preview-course option for demos when API credits are limited.

---

## Live Features

### Course Creation

- Form-driven layout generation with model selection (Groq / Gemini)
- Chapters, topics, and durations generated from AI
- Optional preview course creation (password-protected)

### Step-by-Step RAG Builder

- SSE live log window for chapter generation
- Retrieval-augmented context from previous chapters
- Per-chapter customization instruction

### All-at-Once Generation

- One-click full content generation
- Live build window with status logs
- Error notifications with guidance to preview courses or RAG mode

### Quizzes

- Chapter and final quizzes generated by AI
- Model selection for quiz generation
- Retry with alternate models on API failures

### Integrated Video Learning

- YouTube Data API fetches relevant videos per chapter
- In-app video modal playback (no redirect)

### Progress Tracking

- Mark topics as complete
- Chapter-level progress and overall course progress
- Enrollment-based continuation flow

### Preview Courses (Public)

- Create a public preview course using a password
- Preview courses are highlighted and shown at the top of the dashboard
- Used for demos when API credits are exhausted

### Authentication

- JWT access + refresh token rotation
- Refresh tokens stored as `httpOnly` cookies
- Axios auto-refresh with safe redirect handling

## Key Engineering Decisions

### Why SSE for step-by-step RAG?

SSE provides instant, ordered updates for long-running generation tasks. Users see granular status while chapters are generated and stored.

### Why model selection everywhere?

Users can switch between Groq and Gemini when limits or credits are reached, keeping generation flexible and resilient.

### Why preview courses?

Preview courses are public demo content, allowing reviewers and recruiters to explore the platform when API credits are unavailable.

## Getting Started

### Prerequisites

- Node.js 18+
- MongoDB Atlas
- Groq API key
- Gemini API key
- YouTube Data API key
- Redis (local or Upstash)

### Server

```bash
cd server
npm install
npm run dev
```

### Client

```bash
cd client
npm install
npm run dev
```

Open `http://localhost:5173`
